# opernit

## Synopsis

browser mouse effect using canvas
* circles

## Code Example

```javascript
opernit.circles({
  color: [
    '#fffdb7',
    '#aef4a4',
    '#79b8d1',
    '#e36488',
  ],
  size: 800,
  effectRadius: 40,
  minRadius: 1,
  maxRadius: 3
});
```
